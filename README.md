### Installation
1. Install Go (https://golang.org)
2. Install llir (https://github.com/llir/llvm)
```sh
$ go get -u github.com/llir/llvm/...
```

### How to run
1. build
go to the folder with main.go
```sh
$ go build
```
2. run the execution file
```sh
$ ./iterval ./../test/toy/ll/mod.ll
```
